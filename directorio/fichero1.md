## 3.1.2. Características

El sistema operativo Linux es un sistema operativo multiusuario y multitarea. Desde su creación, ha ido evolucionando hasta convertirse en un sistema operativo totalmente útil, evolucionado y eficiente.

Tiene las ventajas sobre el otro sistema operativo más utilizado, el Windows, en que es libre, configurable, tiene distintos entornos gráficos, etcétera.

## 3.2. Evolución histórica

Los sistemas operativos multitarea usaban el tiempo compartido. Con la llegada de los sistemas operativos multiusuario, la multitarea se repartía entre todos los usuarios del sistema operativo.

Un ejemplo de este tipo de sistema operativo fue el **CTSS**, desarrollado por MIT, como vimos en la Unidad 2. Una de las características de este sistema operativo fue que reducía el tiempo de espera de cualquier proceso.

El sistema operativo **MULTICS** (*Multiplexed Information and Computing Service*, Servicio de información y cómputo multiplexado), se desarrolló por MIT, Bell Labs y General Electric a partir del CTSS. Este sistema operativo estaba pensado para que lo usaran muchos usuarios, estaba escrito en el lenguaje de programación PL/I, y resultó un proyecto demasiado complejo para su época.

A partir de MULTICS, Ken Thompson, trabajador de Bell Labs, desarrolló una versión monousuario, en ensamblador, y la llamó **UNICS** (*Uniplexed Information and Computing Service*, Servicio de información y cómputo uniplexado). Este último concepto se utilizó en oposición a multiplexado en MULTICS, para resaltar que este sistema operativo no era complejo. Posteriormente, al proyecto de desarrollo de UNICS se unieron Brian Kernighan y Denis Ritchie.

UNICS se escribe para el ordenador PDP-11, que tenía protección de memoria por hardware y se escribe en lenguaje B, simplificación del lenguaje BCPL, y que posteriormente Ritchie rediseñaría con el nombre de lenguaje C, que es un lenguaje muy utilizado hasta la actualidad.

El código de UNICS, llamado ya Unix, se reescribe en lenguaje C con lo que se consigue que sea portable, es decir, fácilmente instalable en otros ordenadores, lo que lleva a que se estandarice y se extienda su uso.
